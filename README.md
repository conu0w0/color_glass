# 彩色玻璃 網頁遊戲

一個簡單的彩色玻璃益智拼圖遊戲，可以在網頁上遊玩。

## 遊戲玩法

- 點擊一個格子，選擇要移動的拼圖塊
- 再點擊另一個格子，交換位置
- 目標是讓所有相鄰邊的顏色一致
- 通關後會顯示用時與評語
- 左上角有重置按鈕
- 左側三個按鈕可以切換三種版面

## 使用方式

[點我開始玩遊戲](https://conu0w0.github.io/color_glass/)
